Rounded corners!

The border-radius property added


Specific border radius values can be applied for the border-radius property in the following order: top-left, top-right, bottom-right, bottom-left.

Note!The value of border-radius can also be specified in percentages.

Creating a Circle

A rectangle can be turned into a circle using only CSS.
To create a circle, the border radius should be half of the height and the
width.
